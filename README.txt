# CareerGPS Prototype (Counselor Flow)

This is a lightweight HTML/CSS/JS prototype for the CareerGPS counselor dashboard, student directory, and student profile.

### Features
- Login screen
- Counselor dashboard
- Students list
- Student profile (Overview, Readiness Score, Action Plan, Essays, Tasks, Messages)
- AI-native readiness scoring widgets
- Subscore interactions (drawer behavior)
- Green-themed UI (Mint + Teal hybrid)

### How to Run
Just open `index.html` in any browser — no build step required.

### Structure
- `index.html` → login + main prototype shell
- `styles.css` → theme, layout, components
- `script.js` → drawer logic, tab switching
